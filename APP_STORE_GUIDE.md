# Как опубликовать приложение в Apple App Store

Для публикации iOS-версии приложения вам **обязательно** потребуется компьютер Mac (MacBook, iMac, Mac mini) с установленной программой Xcode.

## Шаг 1: Подготовка

1.  **Компьютер Mac:** Без macOS собрать iOS-приложение невозможно.
2.  **Xcode:** Скачайте бесплатно из Mac App Store.
3.  **Apple Developer Account:** Регистрация стоит $99 в год. Без этого аккаунта вы не сможете опубликовать приложение в App Store. [Зарегистрироваться](https://developer.apple.com/programs/).

## Шаг 2: Сборка проекта

1.  Скачайте код проекта на ваш Mac.
2.  Откройте терминал в папке проекта и установите зависимости:
    ```bash
    npm install
    ```
3.  Соберите веб-версию:
    ```bash
    npm run build
    ```
4.  Синхронизируйте изменения с iOS-проектом:
    ```bash
    npx cap sync ios
    ```

## Шаг 3: Настройка в Xcode

1.  Откройте проект в Xcode:
    ```bash
    npx cap open ios
    ```
    Или откройте файл `ios/App/App.xcworkspace` вручную.

2.  **Настройка подписи (Signing):**
    *   Выберите проект `App` в левой панели (самый верхний пункт).
    *   Перейдите во вкладку **Signing & Capabilities**.
    *   В поле **Team** выберите вашу команду (ваш Apple ID разработчика). Если её нет, добавьте аккаунт через Xcode > Settings > Accounts.
    *   Убедитесь, что `Bundle Identifier` уникален (например, `com.wellness.app`).

3.  **Иконки и заставка:**
    *   В левой панели откройте папку `App` -> `App` -> `Assets`.
    *   Замените `AppIcon` на ваши иконки разных размеров.

## Шаг 4: Тестирование на iPhone

1.  Подключите ваш iPhone к Mac кабелем.
2.  В Xcode в верхней панели выберите ваше устройство вместо симулятора.
3.  Нажмите кнопку **Play** (треугольник) для запуска.

## Шаг 5: Публикация в App Store

1.  **App Store Connect:**
    *   Зайдите на [appstoreconnect.apple.com](https://appstoreconnect.apple.com/).
    *   Создайте новое приложение ("My Apps" -> "+").
    *   Заполните название, язык, Bundle ID (должен совпадать с тем, что в Xcode).

2.  **Архивация (Archive):**
    *   В Xcode выберите в меню **Product > Archive**.
    *   Дождитесь окончания сборки. Откроется окно Organizer.

3.  **Загрузка (Upload):**
    *   В окне Organizer выберите ваш архив и нажмите **Distribute App**.
    *   Выберите **App Store Connect** -> **Upload**.
    *   Следуйте инструкциям мастера (обычно везде "Next").

4.  **Заполнение данных:**
    *   Вернитесь в App Store Connect.
    *   Заполните описание, ключевые слова, загрузите скриншоты (для iPhone 6.5" и 5.5").
    *   Укажите возрастные ограничения и политику конфиденциальности.

5.  **Отправка на проверку:**
    *   В разделе "Build" выберите загруженную сборку (она появится через 10-20 минут после загрузки из Xcode).
    *   Нажмите **Submit for Review**.
    *   Проверка Apple обычно занимает 24-48 часов.

## Важные нюансы

*   **Apple ID:** Убедитесь, что ваш аккаунт разработчика оплачен и активен.
*   **Политика конфиденциальности:** Apple требует ссылку на политику конфиденциальности. Вы можете создать простую страницу (например, в Google Docs или на бесплатном хостинге) с текстом политики.
*   **Функционал:** Apple строго проверяет приложения. Если приложение "пустое" или просто открывает веб-сайт без нативных функций, его могут отклонить (пункт 4.2 App Store Review Guidelines). Capacitor добавляет нативные функции, так что шансы хорошие, но приложение должно быть полезным.
