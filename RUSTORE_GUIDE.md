# Как опубликовать приложение в RuStore

Поскольку ваше приложение написано на React (веб-технологии), для публикации в RuStore его нужно "упаковать" в Android-приложение (.apk или .aab файл). Мы уже настроили проект для использования **Capacitor** — инструмента, который превращает веб-сайт в мобильное приложение.

## Шаг 1: Подготовка на компьютере

Вам понадобится компьютер с установленными инструментами:
1.  **Node.js** (уже установлен, если вы запускаете проект локально).
2.  **Android Studio** (обязательно для сборки Android-приложения). Скачать можно [здесь](https://developer.android.com/studio).

## Шаг 2: Сборка приложения

Выполните следующие команды в терминале в папке проекта:

1.  Соберите веб-версию:
    ```bash
    npm run build
    ```
    Это создаст папку `dist` с готовым сайтом.

2.  Синхронизируйте веб-версию с Android-проектом:
    ```bash
    npx cap sync
    ```
    Эта команда копирует файлы из `dist` в папку `android`.

## Шаг 3: Создание APK/AAB файла

1.  Откройте проект в Android Studio:
    ```bash
    npx cap open android
    ```
    Или запустите Android Studio вручную и откройте папку `android` внутри вашего проекта.

2.  Дождитесь индексации проекта (Gradle Sync).

3.  В меню выберите: **Build > Generate Signed Bundle / APK**.

4.  Выберите **Android App Bundle (.aab)** (рекомендуется для магазинов) или **APK**.
    *   Для RuStore лучше использовать **APK**, если вы хотите просто загрузить файл, или **AAB**, если хотите использовать автоматическое подписание (но RuStore принимает и APK).
    *   *Совет:* Для первой публикации проще выбрать **APK**.

5.  Создайте новый ключ подписи (Key Store), если у вас его нет:
    *   Нажмите **Create new...** под полем "Key store path".
    *   Заполните поля (пароли запомните!).
    *   Заполните Certificate (достаточно имени и фамилии).

6.  Нажмите **Next**, выберите **release** версию и нажмите **Create**.

7.  После завершения сборки Android Studio покажет уведомление. Нажмите **Locate**, чтобы найти файл `.apk` (или `.aab`).

## Шаг 4: Публикация в RuStore

1.  **Регистрация:**
    *   Зайдите на [консоль разработчика RuStore](https://console.rustore.ru/).
    *   Зарегистрируйтесь как физическое лицо (или ИП/Юрлицо).
    *   Вам потребуется подтвердить личность (обычно через ID или паспорт).

2.  **Создание приложения:**
    *   Нажмите "Добавить приложение".
    *   Введите название: **Wellness App** (или ваше название).
    *   Пакет (Package Name): `com.wellness.app` (это мы настроили в конфиге).
    *   Категория: Здоровье / Образ жизни.

3.  **Загрузка версии:**
    *   Перейдите в раздел "Версии".
    *   Загрузите созданный вами `.apk` или `.aab` файл.
    *   Заполните описание ("Что нового").

4.  **Оформление:**
    *   Загрузите иконку (512x512 px).
    *   Загрузите скриншоты (минимум 3 шт, можно сделать на телефоне).
    *   Добавьте описание приложения (о чем оно, функции).

5.  **Отправка на модерацию:**
    *   Нажмите "Отправить на модерацию".
    *   Обычно проверка занимает от 1 часа до 1-2 дней.

## Полезные советы

*   **Иконка:** Чтобы изменить иконку приложения, замените файлы в папке `android/app/src/main/res/mipmap-*`.
*   **Название:** Чтобы изменить название на экране телефона, отредактируйте файл `android/app/src/main/res/values/strings.xml`.
*   **Обновления:** Чтобы выпустить обновление, измените версию в `android/app/build.gradle` (поля `versionCode` и `versionName`), снова соберите APK и загрузите новую версию в консоль RuStore.
